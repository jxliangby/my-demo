数据导出规则

需要导出的表以及对应的字段用--:隔开
其中--:前面为表名，后面为字段名
其中表名和字段名都可用,隔开
对应上述表及字段的导出规则用-->隔开
其中-->前面为需要被替换的，后面为替换成对应的值
样例如下：
api_interface--:target_url
https://www.10.0.3.1:80.com/-->https://test.com/test
https://www.10.0.3.1:83.com/-->https://test.com/test2
api_interface,api_authorization_control--:api_id
crt.cp.apps.appCouponCount-->crt.cp.Apps.AppCouponCount
crv.ewjeb.member.aaaaaaaaaa-->crv.ewjeb.member.AAAAAAAAA
crass.j1eb.tw.head-->crass.j1eb.tw.HEAD
